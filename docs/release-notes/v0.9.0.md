# JeTTY v0.9.0

Polish + hardening: detached-window visual parity, a redesigned Settings panel,
and 25 audit-verified bug fixes.

## Highlights

- **Detached windows, first-class.** Pop-out windows now match the main window's
  corner radius, opacity, CRT effect and caret flash — live with the sliders. No
  more square, opaque pop-outs.
- **Settings, redesigned.** One consistent control language (segmented cyclers,
  real toggle switches, matching steppers/sliders), a calm visual hierarchy, inset
  list wells, a floating theme menu. Verified across dark and light themes.
- **A 60-agent audit, 25 verified fixes.** Every finding was independently
  double-verified before fixing. The ones you'll feel:
  - **PageUp/PageDown work in less/man/vim/htop** (alt-screen aware;
    Shift+PgUp/PgDn always scrolls history).
  - **IME & dead-key input** — accents and CJK commit text now reach the shell.
  - **A huge paste can't freeze the UI** (async PTY writer).
  - Closing the last main tab **no longer kills shells living in detached
    windows** — one is pulled back instead.
  - No more `~` garbage injected right after an F9 summon (X11 synthetic keys).
  - Slow touchpad scrolling isn't dropped anymore; Ctrl+/ sends readline undo;
    modified Home/End/Delete/Insert carry modifiers; atomic config saves; and a
    dozen more lifecycle/render edge cases.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
