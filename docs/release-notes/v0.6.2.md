# JeTTY v0.6.2

A small CRT default tweak.

## Changed

- **CRT curvature defaults to 0** — turning on the CRT effect no longer bows the
  screen by default. Scanlines, shadow-mask, bloom and vignette still apply; the
  barrel/curvature can be turned up under **Settings → Effects**. Only affects
  fresh configs (existing configs keep their stored value).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
