# JeTTY v0.8.0

Detached tabs grew up.

## Highlights

- **Real window chrome for detached tabs.** A detached tab's window now has a
  proper top bar — title, close ✕, **drag to move** — plus edge/corner resizing
  and the bottom perf-HUD strip. (In 0.7.0 the window was bare and couldn't even
  be moved.)
- **Tab context menus.** Right-click a tab → **Detach / Rename / Close Tab**.
  Right-click inside a detached window → **Reattach / Copy / Paste**.
- **Drag & drop.** Drag a tab off the bar to pop it into its own window right at
  the drop point; drag a detached window by its top bar onto the main tab bar to
  reattach. (On platforms without window positioning — e.g. Wayland — moving
  falls back to the compositor drag; no DE-specific code.)

All damage-driven: extra windows cost no idle CPU.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
