# JeTTY v0.4.0

A usability release — JeTTY gets out of your way in TUIs and fits your setup.

## Highlights

- **Missing glyphs now render.** A char the terminal font lacks (e.g. Claude
  Code's `⏵⏵` permission indicator) used to show as a tofu box `□`. JeTTY now
  falls back to a font that has the glyph and draws it — while keeping the
  monospace grid perfectly aligned (like Konsole/Qt).
- **Select & copy inside mouse-driven TUIs.** Hold **Shift** and drag to select
  text even inside apps that grab the mouse (Claude Code, vim, htop, tmux); the
  selection auto-copies on release. The standard terminal convention.
- **Configurable summon hotkey.** Set `summon_hotkey` in
  `~/.config/jetty/config.toml` — `"F12"`, or a chord like `"Ctrl+Shift+F12"`.
  Default `"F9"`.
- **Launch at login.** A new toggle at the bottom of Settings writes an XDG
  autostart entry so JeTTY runs in the background and the summon hotkey is live
  right after you log in. Desktop-environment-independent.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
