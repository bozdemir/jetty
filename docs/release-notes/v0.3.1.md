# JeTTY v0.3.1

A small fix-and-polish release on top of v0.3.0.

## Changes

- **`jetty --version` / `--help`** now print and exit, instead of launching the
  terminal (they used to fall through and open a window).
- **Render hot path** no longer allocates a per-frame spans `Vec` (~0.5 MB at a
  full screen) — the per-cell spans go straight to glyphon as an iterator;
  rendering is byte-identical. `jetty-bench` gained a `cpu prep` / `gpu exec`
  split and a `JETTY_BENCH_GPU=high` selector, which confirm the grid render is
  CPU-prep-bound (text shaping), not GPU-bound.

Thanks to @YKesX for the render-allocation finding and the bench tooling (PR #2).

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
