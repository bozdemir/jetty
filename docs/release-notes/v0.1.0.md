# JeTTY v0.1.0

First public release of JeTTY — a blazing-fast, GPU-accelerated terminal that
summons to the center of your screen (or drops down Yakuake-style) on a global
hotkey. Speed is the #1 priority: ~5.5 ms full-screen frames, ~0% idle CPU.

## Highlights

- **GPU-rendered** via `wgpu` (Vulkan on Linux, Metal on macOS) — full-screen
  frames in ~5.5 ms (well under the 6.9 ms / 144 Hz budget).
- **Global F9 hotkey** — summon from anywhere; first press launches, subsequent
  presses toggle. Center mode and Dropdown (Yakuake/Guake) mode.
- **Five summon shaders** — Phosphor Ignition (CRT power-on), Bayer Crystallize,
  Liquid Drop, Focus Pull, or None.
- **Five themes** — Catppuccin Mocha (default), Tokyo Night, Gruvbox Dark,
  Dracula, Onyx. Every UI surface re-skins with the active theme.
- **Tabs** with rename, confirm-close, and `Ctrl+1–9` quick-jump.
- **Live perf HUD** in the tab bar (ms · fps · CPU% · VT MB/s); goes idle
  automatically with zero extra CPU.
- **Welcome overlay** — neofetch-style splash on first launch.
- **Settings dialog** (`Ctrl+Shift+P`) — all options persisted to
  `~/.config/jetty/config.toml`.
- **macOS/Metal** — full feature parity; build from source (prebuilt `.app`
  is on the roadmap).

## Install

### Linux (prebuilt, no toolchain needed)

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

The script verifies `SHA256SUMS.txt` before installing. Installs to `~/.local/bin`
by default. For a system-wide install:

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | JETTY_PREFIX=/usr/local sudo -E sh
```

### Linux (package / AppImage)

```bash
sudo apt install ./jetty_0.1.0_amd64.deb          # Debian / Ubuntu
chmod +x JeTTY-0.1.0-x86_64.AppImage && ./JeTTY-0.1.0-x86_64.AppImage
```

### macOS (build from source)

```bash
git clone https://github.com/bozdemir/JeTTY.git && cd JeTTY
cargo build --release && ./target/release/jetty
```

## Known issues

- **Resize/p10k prompt scatter** — resizing or changing font size can scatter
  complex prompts (p10k etc.) into fragments. Debounce mitigates; fix in progress.
- **Wayland: no native global shortcut** — use the compositor binding + IPC
  socket workaround in `docs/global-hotkey.md`.
- **macOS: no prebuilt binary** — build from source; prebuilt `.app`/`.dmg` on
  the roadmap.
