# JeTTY v0.6.1

A layout fix for the v0.6.0 Effects tab.

## Fixed

- **Effects settings tab layout** — in the new **Effects** tab, the TINT / COLOR
  RGB sliders and the ANIMATE (roll / flicker / jitter) pills overlapped their
  section headers, so "TINT"+"R" and "ANIMATE"+"ROLL" rendered on top of each
  other. The controls now sit cleanly beside their headers.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/bozdemir/JeTTY/main/install.sh | sh
```

Or a `.deb` / **AppImage** from the assets below (the installer verifies
`SHA256SUMS.txt`). Build from source: `cargo build --release`.
